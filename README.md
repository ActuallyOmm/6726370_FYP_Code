# To run the .ipynb notebooks (on Linux):
1. Create a virtual environment within Python: https://docs.python.org/3/library/venv.html
2. Activate virtual environment
3. Install the required packages using requirements.txt : https://www.geeksforgeeks.org how-to-create-requirements-txt-file-in-python/
4. Open .ipynb notebook of choice and click run all

# To run the .ipynb notebooks in *All* folder:
1. Open Custom All folder and COPY the DATASET folder
2. PASTE the DATASET folder in All directory

# NOTE:

# To use the .pth files:
1. Download file and extract it
2. Then open file until tree structure shown [KDEF, JAFFE, CK+, etc.]
3. Open each folder and copy its contents into the related folder_name/Transfer Learning (i.e., KDEF/Transfer Learning)
